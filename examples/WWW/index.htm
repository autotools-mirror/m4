Current development version is 1.4n.

Development versions contain new features and experiments that might or might not make it into the next official release. The current development version contains among other things (browse the new features for more detail):

  • Uses GNU Automake and GNU Autoconf for configuration.
  • Uses GNU gettext for internationalisation.
  • Support for multiple precision arithmetic in eval.
  • An input syntax table to change how input is parsed.
  • Support for loadable modules.
  • Better control of sync-lines generation.
  • Various bug-fixes.

A new release is expected ready for Spring 2000.

GNU m4 1.4 is from october 1994 and can be considered stable.
